# Tidal Goods Co. Visual Assets

Assets for the BUS 123 rotating case study company Tidal Goods Co.

## Files

- `tidal-goods-wide-simple.png` - recommended 16:9 lesson header image with open space for slide titles.
- `tidal-goods-icon-badge.png` - square badge/icon for worksheets, module labels, and quick visual cues.
- `tidal-goods-wide-detailed-draft.png` - earlier detailed 16:9 concept draft, kept for reference.

## Style Notes

- Coastal retail shop on the North Shore of Massachusetts.
- Palette: teal, navy, sand, weathered wood, and crisp white.
- No embedded readable text, so the images can be reused across slides, Canvas pages, and handouts.
