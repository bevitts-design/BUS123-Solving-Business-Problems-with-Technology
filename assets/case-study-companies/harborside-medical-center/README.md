# Harborside Medical Center Visual Assets

Assets for the BUS 123 rotating case study company Harborside Medical Center.

## Files

- `harborside-medical-center-wide-simple.png` - 16:9 lesson header image with open space for slide titles.
- `harborside-medical-center-icon-badge.png` - square badge/icon for worksheets, module labels, and quick visual cues.

## Style Notes

- Community hospital in Newburyport, Massachusetts.
- Palette: clean white, medical blue, soft teal, muted gray, and a small warm accent.
- No embedded readable text, so the images can be reused across slides, Canvas pages, and handouts.
