# Meridian Advisory Group Visual Assets

Assets for the BUS 123 rotating case study company Meridian Advisory Group.

## Files

- `meridian-advisory-group-wide-simple.png` - 16:9 lesson header image with open space for slide titles.
- `meridian-advisory-group-icon-badge.png` - square badge/icon for worksheets, module labels, and quick visual cues.

## Style Notes

- Financial advisory and bookkeeping firm in Beverly, Massachusetts.
- Palette: dark navy, warm gold, soft white, muted slate, and light coastal blue.
- No embedded readable text, so the images can be reused across slides, Canvas pages, and handouts.
