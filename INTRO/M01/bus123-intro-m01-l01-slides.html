Installing Microsoft 365

07

Follow these steps on your personal device. This takes about 10 minutes.

01 Go to office.com in your browser
02 Click Sign In — use your Endicott email and password

06 Open Excel — confirm it says "Licensed to [Your Name]" under your account
Part Two
Connecting Endicott OneDrive

Connecting OneDrive

09

OneDrive should install automatically with Office 365. Here is how to confirm it is active and syncing.

01 Search for OneDrive in your Start menu (Windows) or Applications (Mac) — open it
02 Sign in with your Endicott email — not a personal Microsoft account
03 Accept the default folder location — click Next through all prompts
04 Look for the blue cloud icon in your taskbar (Windows) or menu bar (Mac) — this means it's running
05 A green checkmark on the icon = synced. A spinning icon = syncing in progress. A red X = needs attention.
⚠️

If it asks you to choose between a personal and work/school account — always choose Work or School. Personal accounts do not connect to Endicott storage.

Saving Files Correctly

14

Two ways to save a file to OneDrive from Excel or Word:

Method 1 File → Save As → OneDrive — Endicott College → navigate to your BUS123 folder
Method 2 Drag the file directly into your OneDrive folder in File Explorer (Windows) or Finder (Mac)
Verify Look for the green sync checkmark on the file — this confirms it is saved to the cloud
Auto-Save in Excel
Turn Auto-Save ON

When a file is saved to OneDrive, Excel shows an Auto-Save toggle in the top left. Turn it on. Your work saves continuously — no more "did I save that?" moments.

Never Do This

Save to Desktop → work on it → "I'll move it later." Later never comes. Save to OneDrive from the first click.

Access From Any Device

15

One of OneDrive's most powerful features: your files are accessible everywhere — no USB drive, no emailing yourself.

  • Personal laptop: Files sync automatically when OneDrive is running
  • Campus lab computers: Go to onedrive.live.com → sign in with Endicott email
  • Phone or tablet: Download the OneDrive app → sign in → view and edit files
  • Any browser, anywhere: office.com → sign in → full Excel, Word, PowerPoint in browser
🌐

Left your laptop at home the day something is due? That is no longer an excuse. Your OneDrive is accessible from any internet-connected device on the planet.

Common Mistake

16
⚠️ CONFUSING "SAVED TO ONEDRIVE" WITH "SYNCED TO ONEDRIVE"

Saving a file to your OneDrive folder is not instant. The file still needs to sync to the cloud — this requires an internet connection and a few seconds.

Risky Behavior

Saving a file at 11:59pm for a midnight deadline, then immediately closing your laptop before the sync checkmark appears. The cloud copy may not have the latest version.

Safe Behavior

Save early, leave OneDrive open, watch for the green checkmark on the file before closing your laptop. Or use Auto-Save — it syncs continuously as you type.

Local vs. Cloud Storage

17
Local Storage (Desktop / USB)
  • Only accessible on that one device
  • Lost if device is stolen, broken, or reset
  • No version history — overwrite = gone
  • Cannot collaborate in real time
  • Must manually back up
OneDrive (Cloud)
  • Accessible from any device, anywhere
  • Protected against hardware failure
  • Automatic version history — restore any prior version
  • Real-time co-authoring with classmates
  • Syncs automatically — no manual backup

Version History

18

OneDrive automatically saves previous versions of every file. This means you can undo changes made days ago — even after closing the file.

01 Open your file in Excel → click File → Info → Version History
02 A list of every saved version appears — with timestamps and who made changes
03 Click any version to open it → click Restore to go back to that state
🔄

"I accidentally deleted all my work" is a solved problem with OneDrive. Version history goes back 30 days automatically, no setup required.

Real-World Skill

In a professional setting, version history protects you when a client asks "can we go back to the version from Tuesday?" — with OneDrive, the answer is always yes.

Quick Troubleshooting

20

Most OneDrive problems fall into three categories:

Red ✕ Sync error — right-click the OneDrive icon in taskbar → click the error message for details. Usually a file name issue (special characters or too long).
Spinning Still syncing — not a problem. Wait for internet connection or give it a moment. Do not close your laptop yet.
No Icon OneDrive not running — search for OneDrive in your apps and open it manually. Set it to run on startup to prevent this.
Wrong Account Right-click OneDrive icon → Settings → Account → Unlink this PC → re-sign in with your Endicott email.
