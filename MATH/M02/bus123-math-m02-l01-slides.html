Two-Step Equations

Reverse the Order of Operations

Level 2 · Reverse order

Standard form:  ax + b = c

01
Undo addition or subtraction (move b to the right side)
02
Undo multiplication or division (divide by a)

Tip: like removing shoes before socks — reverse the order it was built.

// General form
ax + b = c

Step 1: − b both sides → ax = c − b Step 2: ÷ a both sides → x = (c − b) / a

Break-Even Guest Count Between Two Vendors

Vendor Showdown
Vote first: which vendor wins for a larger guest count?
Vendor A
$500 + $25/guest
Vendor B
$200 + $40/guest
A B Depends
500 + 25g = 200 + 40g

− 25g both sides 500 = 200 + 15g − 200 both sides 300 = 15g ÷ 15 both sides g = 20 guests
Watch Out

Moving Variables to the Wrong Side

Spot the Sign Error · 45 seconds
Where does this solution get harder than it needs to be?
Given:   500 + 25g = 200 + 40g

Messy path:   − 40g both sides →   500 − 15g = 200   →   −15g = −300   →   g = 20 ✓

Cleaner move:   subtract the smaller variable term so the coefficient stays positive. Fewer negative signs means fewer arithmetic traps.

Treat Other Variables Like Numbers

Level 4 · Rearrange the formula

A literal equation has multiple variables. Solving for one means treating all others as constants — same rules apply.

Profit Formula
P = R − C  →  solve for R:  R = P + C
Same formula, different variable isolated.
// Solve Revenue = VC + FC for FC

R = VC + FC

− VC both sides FC = R − VC // FC is now isolated

Key Takeaways

Three Rules That Cover Everything

01
Equations are balanced scales. Every operation applied to one side must be applied to both. No exceptions.
02
Collect before you solve. When the variable appears on both sides, move all variable terms to one side first — then it's a two-step.
03
Literal equations are just formula rearrangements. Treat every other letter as a number and apply the same inverse operation rules.
