Markup: Turning Cost Into Price

The first job is knowing what base the percentage is using.

5 / 23
Markup terms

Keep the base clear before calculating.

COST

What Anchor & Oak pays for food, rentals, supplies, or services.

SELLING PRICE

What the customer pays for the package, ticket, or item.

GROSS PROFIT

The difference between selling price and cost before overhead.

6 / 23
Excel first

Markup builds up from cost.

=Cost*(1+MarkupRate)Cost $18.00; markup rate 45%; selling price $26.10.

Manual math: $18.00 + ($18.00 x 45%) = $18.00 + $8.10 = $26.10.

Markdown pressure

Late demand changes the pricing question.

Demand

Will enough seats fill before the event?

Timing

How late is the promotion or discount?

Waste

What supplies or capacity expire if unsold?

10 / 23
Excel first

Markdown moves down from the original price.

=OriginalPrice*(1-MarkdownRate)Original price $52.00; markdown rate 20%; sale price $41.60.

Manual math: $52.00 - ($52.00 x 20%) = $52.00 - $10.40 = $41.60.

Part 3 of 3

Break-Even

How many sales does it take before profit begins?

14 / 23
Break-even terms

Separate costs that stay fixed from costs that move.

FC

Fixed costs: venue, permit, base staffing, insurance.

VC

Variable cost per guest: food, materials, service labor.

S

Selling price per guest or unit.

CM

Contribution margin: selling price minus variable cost.

15 / 23
Excel first

Break-even divides fixed cost by contribution margin.

=FixedCosts/(SellingPrice-VariableCost)Fixed costs $3,600; selling price $48; variable cost $27; break-even 172 guests.

Manual math: $3,600 / ($48 - $27) = $3,600 / $21 = 171.4, rounded up to 172 guests.

Common Mistake

Using the wrong base for the percentage.

Wrong move

Apply markdown to cost, or divide fixed costs by selling price instead of contribution margin.

Better move

Name the base first: cost, original selling price, or contribution margin.
