BUS123 11 / 15
Office Interface

Ribbon, Tabs, & Groups

Ribbon
  • Displays the many features available for your files
  • A collection of tabs organized by function
Tabs
  • Each tab is a collection of groups
  • Tabs can be Standard (always visible) or Contextual (appear when needed, e.g. Table Design)
Groups
  • A collection of related features or commands — e.g. Font, Clipboard, Paragraph
RIBBON HIERARCHY
Ribbon — the entire command interface
↳ Tabs — Home, Insert, Design, Layout…
↳ Groups — Font, Clipboard, Paragraph…
↳ Commands — Bold, Align, Paste…
BUS123 12 / 15
Office Interface

Dialog Boxes, ScreenTips & More

🔧
Dialog Boxes & Task Panes
Not all features show on tabs. Click the launcher icon (↗) in the bottom-right corner of a group to open a dialog box or task pane with additional options.
💬
ScreenTips
Hover over any button or icon to see a ScreenTip — shows the command name, keyboard shortcut, and a brief description.
⚡
Quick Access Toolbar
Sits above the Ribbon. Fully customizable — right-click any command and choose "Add to Quick Access Toolbar" for one-click access.
