# Anchor & Oak Events Visual Assets

Assets for the BUS 123 rotating case study company Anchor & Oak Events.

## Files

- `anchor-oak-events-wide-simple.png` - 16:9 lesson header image with open space for slide titles.
- `anchor-oak-events-icon-badge.png` - square badge/icon for worksheets, module labels, and quick visual cues.

## Style Notes

- Event planning and catering company in Gloucester, Massachusetts.
- Palette: deep green, warm gold, soft ivory, muted autumn tones, and coastal blue accents.
- No embedded readable text, so the images can be reused across slides, Canvas pages, and handouts.
