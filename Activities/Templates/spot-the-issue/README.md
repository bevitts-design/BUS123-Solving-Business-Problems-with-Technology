# Spot The Issue Template

Use this when students need to inspect a worksheet, chart, data table, or AI output and identify problems.

To create a new activity:
1. Copy template.html into the lesson module folder.
2. Rename it using the BUS123 naming pattern.
3. Edit the activityData object.
4. Replace the worksheet preview.
5. Test keyboard navigation.
6. Link it from the course map.
