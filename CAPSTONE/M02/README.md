# BUS123 Capstone M02

This folder contains the public, student-facing AI-assisted investment model audit capstone.

Files:

- `bus123-capstone-m02-l01-project.html`: printable project handout with scenario, AI-use rules, deliverables, and rubric.
- `bus123-capstone-m02-l01-starter.xlsx`: intentionally flawed starter workbook students must audit and repair.
- `source/bus123-capstone-m02-l01-project.md`: editable source for the project handout content.

Instructor-only solution files, error maps, and grading notes belong in the private `BUS123-instructor` repository, not here.
