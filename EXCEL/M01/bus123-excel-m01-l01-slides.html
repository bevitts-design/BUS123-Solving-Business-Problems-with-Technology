Macro-structure

Workbook vs. worksheet

The Workbook

The whole file

What you save and share. HarborsideBudget.xlsx is one workbook.

The Worksheet

The pages inside

The tabs along the bottom — one sheet per department, all in the same file.

Think of it as a binder: one binder (workbook) holding many pages (worksheets). Harborside keeps Cardiology, Radiology, and ER on separate sheets in a single year-end file.

The grid system

Columns, rows & the active cell

  • Columns run vertical, labeled with letters (A, B, C…).
  • Rows run horizontal, labeled with numbers (1, 2, 3…).
  • A cell is the intersection — its address is the column letter then the row number.

Click a cell and it becomes the active cell (the bordered one). On Harborside’s ER sheet, cell B7 holds last month’s visit count — like coordinates in Battleship.

Part 2

Data, Formulas & Functions

From typing numbers to making the sheet do the work
Working faster

Data entry & the Fill Handle

  • Tab moves right, Enter moves down — keep your hands off the mouse.
  • Fill Handle: the small square at a cell’s bottom-right. Drag it to extend a pattern.
  • Double-click the Fill Handle to shoot a formula all the way down a filled column.

Typing Jan then dragging gives Feb, Mar, Apr… — Harborside’s twelve reporting months in seconds.

screenshot: fill handle dragging months
The building blocks

Formula vs. function — the Big Five

A formula is anything you write after = (like =C2+C3). A function is a built-in shortcut (like =SUM(…)). Every formula starts with the equals sign.

SUM

Adds a range. Harborside’s total monthly charges.

AVERAGE

The mean. Average daily ER visits.

COUNT / COUNTA

How many entries? Audit for missing data.

MIN / MAX

Lowest and highest — the busiest and quietest day.

=

Always first. No equals sign, no calculation.

The engine of a smart sheet

Relative, absolute & mixed references

  • Relative (C2) shifts as you copy it down.
  • Absolute ($B$1) locks a cell — the $ sits in front of what it freezes. Press F4 to toggle.
  • Mixed (B$1 or $B1) locks just the row or just the column.

Harborside’s contracted reimbursement rate lives in B1. Lock it so every department uses the same rate.

C2  =  =B2*$B$1
ABC
1Rate0.62
2Cardiology84,200=B2*$B$1
3Radiology61,500=B3*$B$1

Copy down: B2→B3 shifts, $B$1 stays put.

Part 3

Spreadsheets You Can Trust

Clarity, error literacy, and the golden rules
Formatting for clarity

Turn a data dump into a story

Number formats

Accounting for statements, Currency for casual, Percent for payer mix.

Cell hygiene

Auto-fit columns (double-click the border); Wrap Text instead of ultra-wide columns.

Visual hierarchy

Bold headers, bottom borders, banded rows to guide the eye.

Harborside’s payer mix — Medicare 41%, Medicaid 23%, Commercial 28%, Self-pay 8% — reads instantly when formatted as percentages with a clear header row.

Error literacy

What the sheet is trying to tell you

ErrorMeansUsual fix
#REF!Broken referenceA cell the formula needed was deleted or moved.
#DIV/0!Divide by zeroAn empty or zero denominator — e.g. visits = 0.
#NAME?Syntax typoMisspelled function, like =SUMM(…).

An error code isn’t failure — it’s a label pointing you straight to the problem.

The golden rules

Habits of a trustworthy spreadsheet

  • Stop hard-coding. Put a number like the reimbursement rate in its own cell and reference it — never bury it inside a formula.
  • Save constantly. Ctrl + S (Cmd + S) is the cheapest insurance you’ll ever buy.
  • One source of truth. Reference and total automatically instead of copy-pasting the same number around the file.
Common Mistake

Hard-coding the rate inside the formula

Harborside’s reimbursement rate just changed from 0.62 to 0.65…
✗  =B2*0.62
The rate is buried in every cell. Change it and you must hand-edit dozens of formulas — and you will miss one.
✓  =B2*$B$1
Update one cell (B1) and the whole sheet refreshes. Transparent, and safe.

Roughly 88% of spreadsheets contain at least one error — buried hard-codes are a leading cause.

Key takeaways

What to carry forward

  • A workbook holds worksheets; every cell has an address (column letter + row number).
  • Navigate with the Name Box and read the truth in the Formula Bar; a range uses the colon — C2:C5.
  • Reach for a function first — SUM, AVERAGE, COUNT, MIN, MAX — before doing math by hand.
  • Lock shared inputs with $ (F4), read error codes as clues, and never hard-code.
