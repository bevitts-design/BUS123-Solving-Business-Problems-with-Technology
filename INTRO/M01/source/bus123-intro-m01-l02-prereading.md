## What Is Microsoft 365?

Microsoft 365 is a subscription-based suite of productivity software that includes Excel, Word, PowerPoint, OneDrive, Outlook, and Teams. As an Endicott student, you have **full access at no cost** through your college account — which includes installing the desktop applications on up to five personal devices.

## What Is OneDrive?

OneDrive is Microsoft's cloud storage and synchronization service. When you save a file to your OneDrive folder on your laptop, it is automatically copied to Microsoft's servers and made available on every other device connected to the same account.

This means:

- You can never truly "lose" the file to a dead laptop — it lives in the cloud.
- Every device stays current — edit on your phone and the changes appear on your laptop.
- Sharing is a link, not an attachment — colleagues can see the live version, never a stale copy.
- Version history is automatic — OneDrive saves prior versions so you can roll back mistakes.

> **Endicott-Specific Detail**
>
> Your Endicott College Microsoft account includes **1 TB of OneDrive storage**. This is separate from any personal Microsoft account you may already have. Always confirm you are signed in to your **@mail.endicott.edu** account — not a personal Hotmail or Outlook.com account — before saving course files.

## Installing and Activating Microsoft 365

If you have not already installed the desktop applications, follow these steps using your Endicott student credentials:

1. Go to **portal.office.com** and sign in with your @endicott.edu email address.
2. Click **Install apps** in the top right corner and select **Microsoft 365 apps**.
3. Run the downloaded installer and follow the on-screen prompts.
4. Open any Office app (Excel, Word, or PowerPoint), go to **File → Account**, and confirm your Endicott email appears under **User Information**.
5. If you see a different account, click **Sign Out**, then **Sign In** with your @endicott.edu address.

> ⚠️ **Common Mistake**
>
> Students who already have a personal Microsoft account often find that OneDrive syncs to the wrong account. Before saving any course file, verify that the account shown in **File → Account** is your **@mail.endicott.edu** address — not a personal account. Files saved to a personal OneDrive are not accessible to your instructor.

## File Naming Convention

Every file you save in BUS123 must follow the course naming convention. This makes it easy to identify, sort, and share your work without confusion.

**Format:** `LastName_FirstName_BUS123_ModuleN.xlsx`

**Example:** `Evitts_Bethany_BUS123_Module01.xlsx`

> ✅ **Why This Matters**
>
> Instructors receive dozens of files with names like `homework.xlsx` or `final_FINAL_v3.xlsx`. A properly named file tells your instructor exactly who submitted it and what it contains — before it is even opened. This is a professional habit that carries into every workplace.

## Saving, Syncing, and Verifying a File

Saving a file to your OneDrive folder is not the same as the file being available in the cloud. There is a brief delay between saving locally and the file completing its upload to Microsoft's servers. **Always verify sync status before submitting.**

| Sync Icon             | What It Means                        | What to Do                                      |
|-----------------------|--------------------------------------|-------------------------------------------------|
| 🔵 Blue circle/arrows | File is currently uploading          | Wait — do not submit yet                        |
| ✅ Green checkmark    | File is fully synced to the cloud    | Safe to copy share link and submit              |
| ❌ Red X              | Sync error — file did not upload     | Check your internet connection and re-save      |

## Key Vocabulary

| Term                      | Definition                                                                                                                                                                  |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Microsoft 365**         | A subscription-based suite of productivity applications including Excel, Word, PowerPoint, OneDrive, Outlook, and Teams. Endicott students have full access at no cost through their college credentials. |
| **OneDrive**              | Microsoft's cloud storage and file synchronization service. Files saved to OneDrive are automatically backed up to remote servers and accessible across all signed-in devices. |
| **Cloud Storage**         | Files stored on remote servers accessed over the internet rather than on a physical hard drive. Enables multi-device access and protects against local hardware failure.       |
| **Sync**                  | The process of ensuring the local copy of a file and the cloud copy are identical and up to date. Indicated by a green checkmark on the file icon in your OneDrive folder.    |
| **Desktop Application**   | A software program installed and running directly on your computer, as opposed to a browser-based version. Desktop apps have full features and work offline.                  |
| **File Naming Convention**| A standardized format for naming files so they are identifiable, sortable, and consistent. BUS123 format: `LastName_FirstName_BUS123_ModuleN.xlsx`.                          |
| **Activation**            | The process of signing in to Microsoft 365 with your Endicott credentials to unlock full access to the desktop applications. Verified under File → Account in any Office app. |
| **Version History**       | OneDrive's automatic record of prior saved states of a file. Allows you to restore an earlier version if content is accidentally deleted or overwritten.                      |
