Formatting Is Communication

Before we touch the ribbon, we need a standard for what a good worksheet does for its reader.

Why Formatting Matters

EXCEL-M02-L01 · 06
Guide attentionHeaders, totals, and spacing tell the reader where to start.
Signal reliabilityConsistent formats make the data feel handled with care.
Speed decisionsA manager can compare, scan, and act without decoding the sheet first.

Formatting is not cosmetic. It is the difference between data someone can act on and data someone has to interpret.

Excel Syntax: Formats

EXCEL-M02-L01 · 12

Step 1: select the cells you want to format. Excel cannot format what you have not selected.

Home tab - Number group - dropdown - Currency or Accounting // Keyboard shortcut for Currency format: Ctrl + Shift + $ // Keyboard shortcut for Percent format: Ctrl + Shift + % // Keyboard shortcut for Comma / Number format: Ctrl + Shift + !

Currency vs. Accounting: Currency places the symbol next to the number. Accounting aligns symbols and negatives for financial statement readability.

Audit The Handoff

Professional worksheets stay useful after the first screen, first scroll, and first sort.

Banded Rows

EXCEL-M02-L01 · 15

Banding helps the eye track across wide tables. Use it quietly, then reserve stronger emphasis for totals.

  • Detail rows use subtle alternation.
  • Total rows use stronger border and weight.
  • Headers stay consistent across the workbook.

Freeze Panes

EXCEL-M02-L01 · 16

Anchor & Oak tracks 80 events a year. When students scroll to row 60, the labels should still be visible.

View tab - Freeze Panes - Freeze Top Row // Locks row 1 while you scroll down. Click cell B2, then View - Freeze Panes - Freeze Panes to freeze both row 1 and column A.

Unfreeze: same menu. The option changes to "Unfreeze Panes" once a freeze is active.

Common Mistake

EXCEL-M02-L01 · 18

Do not merge cells inside a data table.

Merged cells look tidy on screen but break sorting, filtering, and copy-paste. Use Center Across Selection when the label needs to span columns visually.

Key Takeaways

EXCEL-M02-L01 · 20
  1. 1Formatting communicates meaning. It tells the reader what kind of fact each value represents.
  2. 2Consistency creates trust. Headers, totals, alignment, dates, and money columns should follow one visible logic.
  3. 3Good worksheets survive use. They scroll, sort, filter, and update without breaking the reader's confidence.
