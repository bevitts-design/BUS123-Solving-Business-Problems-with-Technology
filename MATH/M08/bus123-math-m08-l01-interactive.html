Find the cost of time before opening Excel.

Start with the same short example students will rebuild in the workbook: $1,500, 7%, and 3 years. Then compare simple interest with compounding and name who benefits from the difference.

Predict First

If $1,500 grows for 3 years at 7%, which future value feels more reasonable?

What Changed?

Simple interest earns interest only on the starting principal. Compound interest earns interest on prior interest too.

Build in Excel

Make the Contrast

The compound formula uses an exponent, so each year can grow from the prior year's ending value.
