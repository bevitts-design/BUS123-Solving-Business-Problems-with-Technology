Before Spreadsheets

The Ledger Era

One mistake. Start over.
Paper ledgers required accountants to recalculate every affected number by hand when a single entry changed. A single-formula update in Excel does in a second what took hours.
Excel automated the logic — but only if the references are right.
That's the core lesson today. Excel is extraordinarily powerful — and brittle when references are wrong. You're learning to use it the way professionals do.

Your Two Most Important Boxes

Name Box + Formula Bar

Name Box
Tells you where you are
Shows the address of the selected cell — A1, B3, C10. Click it to jump directly to any cell by typing an address.
Formula Bar
Tells you what's really there
The cell might display $14.99, but the formula bar shows whether that's a typed value or a formula. Always check here before editing.
// Cell C3 selected
Name Box: C3
Formula Bar: =B3*$B$1
// Cell displays: 20.99
The cell shows the result.
The bar shows the formula.
Tidal Goods Co. — 500-Row Catalog

Navigate Large Files Like a Pro

Ctrl + End
Jump to Last Cell
Jumps to the last used cell in the worksheet — the bottom-right corner of your data. Instant orientation in any file.
Ctrl + Home
Jump to A1
Returns you to the beginning. Standard first move when opening a file you didn't build yourself.
Ctrl + Arrow
Jump to Data Edge
Ctrl+Down jumps to the last filled cell in a column. Ctrl+Right across a row. Skips over empty cells and stops at the boundary.

Formatting Is Communication

Four Things Formatting Tells Your Reader

01
Data type: Is this a dollar amount? A percentage? A date? Number format makes it unambiguous.
02
Structure: Bold headers, borders, and shading show where data begins and how it's organized.
03
Alignment: Numbers align right. Text aligns left. This is a professional convention — not optional.

Common Mistake

Skipping Number Format

⚠️
0.15 is not the same as 15%
You type 0.15 for a 15% discount rate. The cell shows 0.15. A reader doesn't know if that's 15 cents, 0.15 of something, or 15%. Number format is not decoration — it's part of the data. Always apply it before sharing any file. Home tab → Number group → choose Currency, Percentage, or Number.

Cell References
Relative · Absolute · the F4 shortcut · when to use each
The Core Concept

Two Kinds of Cell References

Relative — A1
Moves when you copy
A relative reference updates automatically when you copy or fill the formula to new cells. Copy a formula from row 2 to row 3 — every relative reference shifts down one row. This is usually what you want for row-by-row calculations.
Absolute — $A$1
Stays put when you copy
Dollar signs lock the reference. $A$1 always points to cell A1 — no matter where you copy the formula. Use this for rates, tax percentages, markup %, or any single shared value that every formula needs to reference.

Why Absolute?
$B$1 never moves
If we used B1 (relative), copying the formula to row 4 would shift the reference to B2 — not the markup rate. $B$1 locks it. Copy to 500 rows: every formula still reads the 40% in B1.
Why Relative for B3?
Cost moves down with each row
B3 is the cost for row 3. When we copy down, we want it to become B4, B5, B6 — each product's own cost. Relative reference does exactly that.
Formula Syntax

Reference Types Side by Side

// Relative — moves when copied
=B3 * (1 + B1) ← B1 shifts to B2, B3... when copied down
// Absolute — locked in place
=B3 * (1 + $B$1) ← $B$1 always reads the rate in B1
// Mixed — column locked, row relative
=B3 * (1 + $B1) ← column B is locked; row can shift
Press F4 after typing a cell address → cycles through all four modes
The Analyst's Shortcut

F4 Cycles Through Reference Modes

1×
$A$1 — both column and row locked. The most common absolute reference. Use for a single shared value (rate, tax, markup %).
2×
A$1 — row locked, column free. Useful when building lookup tables that stretch across columns.
3×
$A1 — column locked, row free. Useful when building tables that extend down rows.
Back to start
4× → A1 — relative again. One more press and you're back where you started. Professional analysts use F4 constantly — never type dollar signs by hand.

Key Takeaways

Three Things to Take With You

01
The Name Box tells you where you are. The Formula Bar tells you what's really in a cell. Check these two boxes before you do anything else.
02
Formatting is communication, not decoration. Apply number format before sharing any file — 0.15 and 15% are not the same thing to your reader.
03
Dollar signs lock. Use $B$1 when every formula needs to reference the same cell. Press F4 to cycle — never type dollar signs by hand.
